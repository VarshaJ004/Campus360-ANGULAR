# AngularJS Practice Project

## Overview

This project is created **for learning and practice purposes only**. It is intended to help understand the basic concepts and core features of **AngularJS (1.x)**.

## Concepts Practiced

The project demonstrates the following AngularJS concepts:

- AngularJS Module
- Controllers
- Two-way Data Binding (`ng-model`)
- Expressions
- `ng-repeat`
- `ng-click`
- `ng-class`
- `ng-style`
- `ng-disabled`
- `ng-readonly`
- `ng-switch`
- `ng-init`
- Filters (`uppercase`, `lowercase`, `currency`, `date`)
- Form Validation
- Form State Properties (`$valid`, `$invalid`, `$pristine`, `$dirty`, `$touched`)
- Input Validation using `required` and `ng-pattern`

## Purpose

The main purpose of this project is to:

- Practice AngularJS fundamentals.
- Understand how AngularJS directives work.
- Learn form handling and validation.
- Explore data binding and user interaction.
- Build a strong foundation before working on larger AngularJS applications.

## Note

This is **not a production-ready application**. It is a practice project created to implement and experiment with AngularJS concepts and basics while learning the framework.
